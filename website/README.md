# WeaveFlux Website

This directory contains the static product page for WeaveFlux.

## Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

```powershell
cd website
python -m http.server 8080
```

## Deploy

The page is ready for GitHub Pages. Use `website/` as the publishing source, or copy its contents to the Pages branch/folder configured for the repository.

The primary download link points to:

```text
https://github.com/glosc-ai/WeaveFlux/releases
```
