为移动创作而做

把视频生成流程收进一个本地 App。

WeaveFlux 聚焦轻量、私密、可替换端点的 AI 视频工作流。它不维护账号系统， 不托管你的作品，也不把 API 凭据交给自建服务；你配置兼容 OpenAI 结构的服务后即可开始创作。

本地优先

你的端点、密钥和作品由你掌控。

01

无自建远程服务

应用直接从本机向你配置的 OpenAI-compatible endpoint 发起请求。

02

BYO Key 模式

Base URL、API Key 和模型由用户自行提供，适合接入不同兼容服务。

03

本地加密保存

敏感配置通过 Android KeyStore 相关能力加密保存，避免明文落盘。

配置一次，然后专注生成。

FAQ

常见问题

WeaveFlux 是否需要注册账号？

不需要。当前产品设计不包含账号、会员或充值流程。

API Key 会上传到哪里？

不会上传到 WeaveFlux 的服务器。应用没有自建远程服务，凭据只用于从本机请求你配置的模型端点。

可以使用自己的模型服务吗？

可以。只要服务兼容 OpenAI 风格的视频生成接口结构，就可以在设置页填写 Base URL、API Key 和模型名称。

生成的视频保存在哪里？

临时和本地作品优先保存在应用私有目录；导出到系统相册时使用 Movies/WeaveFlux 目录。
