## UI/UX 设计文档

### 1. 视觉风格与全局规范

* **设计主题：** 极简工业科技风 (Minimalist Tech Noir)
* **全局色彩调色板：**
* 底色 (Background): `#0f172a` (深邃夜幕蓝，替代纯黑，减少视觉疲劳)
* 表面色 (Surface/Card): `#1e293b` (暗岩灰，用于输入框、卡片背景)
* 主强调色 (Primary Accent): `#10b981` (极光绿，用于生成大按钮、核心状态提示)
* 次强调色 (Secondary Accent): `#3b82f6` (科技蓝，用于设置、参数滑块、轮询进度)
* 文本主色: `#f8fafc` (近纯白)；文本次色: `#94a3b8` (板岩灰)


* **圆角规范：** 统一采用大圆角剪裁（卡片 `16dp`，按钮 `12dp`，输入框 `8dp`）。

### 2. 核心页面结构与交互说明

#### P1：主创作舱 (Creation Workspace)

* **布局：** 单页流式布局。顶部为药丸形（Pill Shape）分段选择器（文生视频 / 图生视频）。
* **内容区：** 高度 `150dp` 的暗岩灰卡片输入框。支持大文本提示词输入。在图生视频模式下，此处变为带虚线边框的 `+ 选择参考图片` 区域。
* **高级控制台（可折叠）：** * 模型指定文本输入框（默认 placeholder 显示 `"t2v-default-model"`）。
* 长宽比按钮组：三个并排的微缩矩形图表（16:9 横屏、9:16 竖屏、1:1 方块）。
* 控制滑块：动态幅度 (Motion Scale)，使用科技蓝作为滑块填充。


* **底部固定栏：** 常驻一个高 `54dp` 的“极光绿”高光渐变按钮，文字为“开始织影”。当未配置 API Key 时，该按钮呈现灰色，点击弹出底层引导去设置。

#### P2：异步任务队列 (Task Orbit)

* **布局：** 精简卡片列表流。
* **生成中卡片：** 左侧展示用户输入的 Prompt 缩略图。右侧展示一个环形无限旋转的科技蓝微缩进度条，并标注 `Go 核心正在轮询中...`。
* **失败卡片：** 卡片边缘呈现微弱的暗红色调，右侧显示警示图标，点击卡片可展开查看原生 OpenAI 错误码（如 `401` 或 `429`），底部提供“重试”按钮。

#### P3：私密画廊与全屏预览 (Private Gallery)

* **画廊主页：** 采用经典两列瀑布流（Staggered Grid）。每个格子直接渲染本地沙盒视频的第一帧作为封面。
* **全屏沉浸播放页：** 点击画廊格子，全屏黑底播放。底部浮动两个大图标：`[下载至系统相册]`（带向下的箭头动效）和 `[删除本地资产]`（带二次确认浮窗）。

#### P4：端点设置面板 (Endpoint Configurations)

* **布局：** 包含两个核心输入框：`Base URL`（例：`https://api.glosc-ai.one/v1`）与 `API Key`（默认隐藏显示为 `••••••••••••`）。
* **连通性测试：** 输入框下方配有一键“测试连接”按钮。点击后，按钮内部变为加载状态，向输入的端点发送一个廉价的 `/v1/models` 查询。成功则提示绿色的“通信成功”，失败则显示红色的错误详情。
